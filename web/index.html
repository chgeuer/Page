Was ist eine Programmiersprache

Der Computer kann nicht einfach so Programmiersprachen, wie Java verstehen und umsetzen. Es wird ein Kompiler benutzt, welcher Sprachen, wie C# oder Java in Maschinencode kompiliert. Dieser Maschinencode besteht aus 0 und 1, welche für der Computer verständlich ist. Daher werden Progrmmiersprachen benutzt, welche ein Mensch gut verstehen kann und dank einem Kompiler der Computer auch. Programmiersprachen, welche sehr nah an Maschinencode sind, sind Sprachen wie Assembler oder C. Der Programmierer schreibt mit Hilfe einer Programmiersprache Quellcode, welcher in Maschinencode kompiliert wird.

C#

Bei C# handelt es sich um eine Objektorientierte Programmiersprache, in welcher man sehr viel mit Klassen und Objekten arbeitet. C# ist sehr ähnlich zu der bekannten Programmiersprache Java, bei welcher es sich auch um eine Objektorientierte Programmiersprache handelt. C# wurde um das Jahr 2000 herum von Microsoft entwickelt und ist ein Teil von dem .NET Framework. Außerdem gehört C# wie der Name schon sagt der der Familie von C. Mit Hilfe von C# lassen sich Webanwendungen für Server bauen, Spiele programmieren als auch Programme für den eigenen Computer bauen.

F#

F# ist eine Multi-Paradigmen-Sprache, welche einen starken Fokus aus der funktionalen Programmierung hat. F# wurde ebenfalls von Microsoft entwickelt und gehört ebenfalls zu dem .NET Framework. F# ist im Jahr 2002 erschienen und wurde von Don Syme designt. F# wird besonders oft bei sehr kompliziertem Code verwendet, bei welchen sich ein Team keine Fehler leisten kann. Es lassen sich jedoch wie in C# mit Objekten und Methoden arbeiten. Es lassen sich auch mit F# Programme für Webanwendugnen bauen, wie mit C#, jedoch welche man für sich wählt ist Präferenzsache.

JavaScript

JavaScript ist eine Scriptsprache und wurde ursprünglich für dynamisches HTML in Webbrowsern von Netscape im Jahr 1995 entwickelt. Es wird oft zusammen mit HTML und CSS verwendet und man sagt JavaScript macht die ,,Organe" einer Webseite wohingegen HTML die Knochem sind und CSS die Haut. Der Designer und Entwickler von JavaScript ist Brendan Eich. Eine Sprache, welche sehr aktuell und beliebt ist, ist Typescript, welche sich sehr ähnlich anfühlt wie JavaScript. Es lassen sich aber auchmit Hilfe von NOTE.js wie bei C# Programme bauen welche auf einem Computer laufen

Java

Java ist ebenfalls, wie C# eine Objektorientierte Programmiersprache und es lassen sich auch die selben Projekte umsetzten, wie mit C#. Java basiert ebenfalls, wie C# auf C. Java ist jedoch viel verbreiteter als C# und welche der Sprachen man bevorzugt ist Präferenzsache.Java ist im Jahr 1995 erschienen und wurde von James Gosling und Sun Microsystems designt. Entwickelt wurde Java von Sun Microsystems und Oracle. Man darf jedoch nicht Java mit JavaScript verwechseln. Die beiden Sprachen haben nichts miteinander gemeinsam außer einen ähnlichen Namen.

Phyton

Python ist eine aktuell sehr beliebte Scriptsprache und erschien im jahr 1991. Designt wurde Python von Guido van Rossum. Python basiert wie viele andere Programmiersprachen auf C und C++. In Python benutzt man keine ; oder {} sondern arbeitet sehr viel mit Tabs und Abständen. In Python hat der Entwickler sehr viele Freiheiten. Mit Phython kann man besonders gut Maschine Leaning umsetzten also ein Algorithmus, welcher durch starke Wiederholung z.B. Menschen erkennen kannn oder ein Nutzerverhalten analysieren kann.
